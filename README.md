UX-TV
---
 
Hello! :)

UX-TV, made with ReactJS.


Usage
---
 
Start the development server with this command:
 
```
npm start
```


Setup
---
 
```
npm install
```


Compile
---
 
```
npm run compile
```


Test
---

```
npm test
```
